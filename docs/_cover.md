# Meadow-Connection-Meadow-Endpoints

> Meadow connection that fronts a remote meadow-endpoints REST API as a meadow data source. Pairs with the in-meadow Meadow-Provider-MeadowEndpoints provider — this module supplies the connection-manager wrapper, form schema, and shared cookie/header state so consumers (retold-databeacon, retold-data-service, ...) can pick `Type: 'MeadowEndpoints'`.
